# 基于 LLM 的消费者评论情感分析与购买建议智能体

## 1 研究背景

消费者评论是电商平台和产品运营中的重要数据来源。大量评论包含用户对产品质量、价格、物流、售后和使用体验的真实反馈。人工阅读评论效率低，传统关键词统计又难以形成自然语言购买建议。因此，本项目构建一个基于大语言模型的消费者评论分析智能体。

## 2 智能体定义

本智能体不是日常办公工具，也不是娱乐产品，而是面向消费者决策和商家运营分析的专业应用。系统输入评论 CSV 数据，输出情感分析、关注点识别、商品风险排序和购买建议。

## 3 技术路线

系统采用“LLM + 工具 + 工作流”的结构：

1. 本地 JavaScript 工具解析 CSV。
2. 规则工具根据评分和关键词完成情感初判。
3. 统计工具计算关注点分布、负面率和商品风险。
4. 阿里云百炼 Qwen 模型根据结构化统计结果生成专业报告。
5. Web 页面展示图表、表格和购买建议。

## 4 系统功能

- 评论数据输入
- 情感分布统计
- 产品质量、价格价值、物流包装、售后服务和功能体验识别
- 商品负面率排序
- Qwen 生成自然语言分析报告
- Qwen 生成购买建议
- JSON 报告下载

## 5 结论

该系统证明了大语言模型在消费者评论理解和购买辅助决策中的应用价值。通过将确定性统计工具与 LLM 语言理解能力结合，智能体既能保证基础数据分析的稳定性，也能生成更符合用户阅读习惯的专业建议。
